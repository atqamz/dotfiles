# Context discipline

- Keep work in chunks that fit ~100K tokens. Finish one self-contained chunk, then stop and recommend `/compact` (continue) or `/clear` (new task) before the next.
- Past ~100K a hook injects a reminder — act on it, don't wait to be asked.
- Don't pre-load context you don't need: read the slice of a file you need, not the whole file; prefer targeted search over broad dumps.
